Quick Start Instructions
========================

Install PGL
-----------
To install Paddle Graph Learning, we need the following packages.


.. code-block:: sh

    paddlepaddle >= 1.4 (Faster performance on 1.5)
    networkx
    cython

We can simply install pgl by pip.

.. code-block:: sh

    pip install pgl
